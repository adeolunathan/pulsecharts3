Dual Save Functionality Test

Test Scenario

This test simulates the dual save functionality:

  • Quick Save (Save) - Updates existing chart without prompts
  • Save As - Creates new chart with name prompt
  • Ctrl+S - Keyboard shortcut for Quick Save

Test Instructions

  1. Click "Save As" to create your first chart (will prompt for name)
  2. Click "Save" to update the existing chart (no prompt)
  3. Press Ctrl+S to test keyboard shortcut
  4. Click "Save As" again to create a new copy
